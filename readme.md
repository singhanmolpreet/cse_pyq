# CSE Previous Year Questions (PYQs) - 2018 Scheme

Welcome to the central repository for **Computer Science and Engineering (CSE) Previous Year Questions**, specifically curated for the **2018 Scheme**. This collection is designed to help students streamline their exam preparation and better understand the question patterns followed by the university.

---

## 🤝 Contribution Guide

This repository is powered by the community. If you have question papers missing from this collection, your contribution would be highly appreciated!

### How to Contribute

1. Fork this repository.
2. Upload the question papers (preferably in **PDF format**) to the appropriate semester folder.
3. Ensure the filename follows this format:

```bash
SubjectName_MonthYear.pdf
```

Example:

```bash
OperatingSystems_Dec2023.pdf
```

4. Create a Pull Request, and it will be reviewed and merged.

### Alternative Method

If you are not familiar with GitHub, you can also:

- Open an issue in the **Issues** tab
- Attach the question papers there
